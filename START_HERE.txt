❓ QUICK TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────────

"Where do I find my serial port?"
   → Windows: Device Manager → Ports (COM & LPT)
   → Linux: ls /dev/ttyUSB* or ls /dev/ttyACM*
   → Mac: ls /dev/tty.usbserial*

"Failed to open serial port"
   → Wrong port OR already in use by another program
   → Solution: Check Device Manager, close other apps

"Web page won't load"
   → Flask server not running
   → Solution: Check terminal where you ran python

"No data in plot"
   → STM32 not sending data OR serial connection wrong
   → Solution: Verify firmware is running, check COM port

"Connection shows Disconnected"
   → Serial port mismatch OR firmware not initialized
   → Solution: Verify port in app.py matches Device Manager

👉 For complete troubleshooting: See DATAVISUALIZATION_TROUBLESHOOTING.md
